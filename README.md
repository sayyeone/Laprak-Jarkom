## Tentang Repositori Ini

Repositori ini berisi kumpulan laporan praktikum mata kuliah **Jaringan Komputer** yang dilaksanakan selama **Semester 4 Tahun Akademik 2025/2026**. Mata kuliah ini membahas konsep dan implementasi jaringan komputer secara praktis, mulai dari analisis protokol jaringan menggunakan Wireshark hingga pembuatan aplikasi jaringan berbasis socket programming dengan Python. Praktikum berlangsung selama 16 pekan dengan total 16 modul yang mencakup berbagai topik jaringan komputer.

---

## Daftar Modul

| Modul | Judul | Tujuan Praktikum |
|-------|-------|-----------------|
| 1 | Running Modul | Mahasiswa mengetahui aturan dan sistem pelaksanaan praktikum serta memastikan tools yang akan digunakan berfungsi dengan baik |
| 2 | Pengenalan Tools | Mahasiswa mengenal dan mampu mengoperasikan Wireshark sebagai tools utama analisis jaringan, termasuk cara menjalankan dan melakukan test run |
| 3 | HTTP | Mahasiswa memahami cara kerja protokol HTTP melalui analisis paket, mencakup HTTP GET/response, conditional GET, pengambilan dokumen panjang, embedded objects, dan autentikasi |
| 4 | DNS | Mahasiswa memahami cara kerja DNS melalui penggunaan nslookup, ipconfig, dan analisis paket DNS menggunakan Wireshark |
| 5 | UDP | Mahasiswa memahami karakteristik dan cara kerja protokol UDP melalui analisis paket |
| 6 | TCP | Mahasiswa memahami cara kerja protokol TCP, termasuk mekanisme transfer data dalam jumlah besar dan congestion control |
| 7 | Socket Programming | Mahasiswa mampu membuat aplikasi jaringan sederhana menggunakan socket programming dengan Python, baik menggunakan protokol UDP maupun TCP |
| 8 | Pembagian Topik Tugas Besar | Mahasiswa mengetahui pembagian topik dan ketentuan pengerjaan Tugas Besar praktikum |
| 9 | Web Server | Mahasiswa mampu membangun dan menjalankan web server sederhana menggunakan Python |
| 10 | IP | Mahasiswa memahami protokol IP melalui analisis paket traceroute, mencakup IPv4 dasar, fragmentasi, dan IPv6 |
| 11 | DHCP | Mahasiswa memahami cara kerja protokol DHCP melalui analisis jejak paket yang tertangkap |
| 12 | ICMP & Asistensi Tugas Besar | Mahasiswa memahami cara kerja protokol ICMP melalui ping dan traceroute, serta mendapatkan bimbingan pengerjaan Tugas Besar |
| 13 | Ethernet and ARP | Mahasiswa mampu menangkap dan menganalisis frame Ethernet serta memahami cara kerja Address Resolution Protocol (ARP) |
| 14 | 802.11 WiFi | Mahasiswa memahami cara kerja jaringan WiFi melalui analisis beacon frames, proses transfer data, serta association dan disassociation |
| 15 | Presentasi Tugas Besar | Mahasiswa mempresentasikan hasil pengerjaan Tugas Besar di hadapan Asisten Praktikum |
| 16 | Asistensi 2 (Praktikum Susulan) | Mahasiswa yang belum memenuhi kehadiran dapat mengikuti praktikum susulan |

## Tools yang Digunakan

- **Wireshark:** Network protocol analyzer untuk menangkap dan menganalisis paket jaringan
- **Python:** Bahasa pemrograman untuk socket programming dan pembuatan web server
